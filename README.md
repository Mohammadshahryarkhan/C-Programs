# Grade Calculator in C

This program takes student marks as input and assigns a grade using if-else conditions.

## Logic
- 90+ → Grade A
- 75–89 → Grade B
- 60–74 → Grade C
- 50–59 → Grade D
- Below 50 → Grade F

##  How to Run
1. Compile using gcc:
   gcc grade-calculator.c
2. Run:
   ./a.out
